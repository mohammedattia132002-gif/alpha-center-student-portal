# Student Portal

بوابة الطالب تعمل حاليًا كتطبيق `React + Vite` مع ربط مباشر بقاعدة `Supabase`.

## المتطلبات

- Node.js 20 أو أحدث
- مشروع Supabase يحتوي على جداول الطلاب والسجلات والامتحانات

## التشغيل المحلي

1. ثبّت الاعتماديات:
   `npm install`
2. انسخ `.env.example` إلى `.env`
3. أضف قيم:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
4. شغّل البوابة:
   `npm run dev`

## البناء والمعاينة

- بناء الإنتاج:
  `npm run build`
- معاينة النسخة المبنية:
  `npm run preview`

## الربط مع Supabase

البوابة تعتمد على Supabase في:

- تسجيل دخول الطالب
- استقبال طلبات الانضمام
- عرض الحضور
- عرض المدفوعات
- عرض الدرجات
- تحميل الامتحانات
- حفظ نتائج الامتحانات

## ملاحظات مهمة قبل الإطلاق

- عند غياب متغيرات Supabase ستستخدم البوابة `localStorage` كمسار احتياطي.
- حالة الامتحان للطالب يتم تحديدها من `exam_results` الخاصة به، وليس من حالة عامة مشتركة بين كل الطلاب.
